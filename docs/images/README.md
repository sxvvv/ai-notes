# 📸 项目截图

这个目录用于存放项目截图。

## 如何添加截图

1. **访问在线演示**
   - 打开：https://ai-notes-h2u49eu67-suxins-projects-d617dd70.vercel.app/

2. **截图建议**
   - `dashboard.png` - 仪表板页面
   - `editor.png` - 编辑器页面
   - `ai-assistant.png` - AI 聊天助手
   - `mobile-view.png` - 移动端视图（可选）

3. **截图工具**
   - Windows: `Win + Shift + S` (截图工具)
   - 浏览器: 开发者工具 → 设备工具栏 → 截图
   - 第三方工具: Snipaste, ShareX 等

4. **截图要求**
   - 格式: PNG 或 JPG
   - 尺寸: 建议宽度 1200-1600px
   - 质量: 清晰，能展示主要功能

5. **添加到 README**
   - 将截图保存到此目录
   - 在 README.md 中使用相对路径引用
   - 例如: `![Dashboard](./docs/images/dashboard.png)`
